Sistema de chat flotante "Voces del Subsuelo" para Radio Vesánico.

== Description ==

Este plugin añade un panel desplegable de comentarios en vivo sobre el reproductor de audio.
Características:
- Estética Post-Punk / Underground.
- No requiere registro (Nombre + Avatar generado).
- Panel de Administración en WP-Admin para borrar comentarios.
- AJAX/jQuery para actualizaciones sin recargar.

== Installation ==

1. Crea una carpeta llamada `vesanico-chat` en `wp-content/plugins/`.
2. Copia el archivo `vesanico-chat.php` dentro.
3. Crea una carpeta `assets` dentro de `vesanico-chat`.
4. Dentro de `assets`, crea `js` y `css`.
5. Copia `assets/js/vesanico-app.js` y `assets/css/vesanico-style.css` a sus carpetas.
6. Activa el plugin desde WordPress.
